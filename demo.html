🎮 Demo-Modus

Diese Anwendung ist ein Prototyp. Alle Daten werden nur lokal in Ihrem Browser (localStorage) gespeichert — nichts wird an einen Server gesendet. Wenn Sie den Browser wechseln oder den Verlauf löschen, sind die Daten weg. Bei der späteren Umstellung auf WordPress werden echte Nutzer- und Datenbankverbindungen eingerichtet.

⚠ Alles zurücksetzen

Löscht alle lokalen Daten im Browser — Konten, Buchungen, Verfügbarkeiten, Abrechnungen. Die Anwendung ist dann wie frisch installiert. Nur diesen Browser betroffen.

💡 Typische Test-Flows:
  1. Als margarete@demo.de einloggen → Mitgliederbereich → Sitzwachen-Kalender → Verfügbarkeit eintragen
  2. Ausloggen, als charite@demo.de einloggen → Schicht buchen → PDF-Bestätigung
  3. Ausloggen, als margarete@demo.de einloggen → Abrechnung → neuer Beleg nach Einsatz
  4. Als vorstand@demo.de → alle Daten einsehen
